ゼロ設定で開始

yt-dlp, FFmpeg, Deno を自動ダウンロード。インストールして起動するだけですぐ使えます。

🌐

1800+ サイト対応

YouTube, ニコニコ動画, TVer, ABEMA, TikTok, X, Bilibili など、あらゆるサイトに対応。

🎥

配信録画モード

ライブ配信をリアルタイムで録画。開始から録画、途中からの録画に対応。

🎬

QuickTime 互換

MP4 は H.264+AAC でエンコード。ダウンロードしてすぐ QuickTime Player で再生。

🔒

Cookie・ログイン対応

アプリ内ブラウザでログインし、会員限定コンテンツもダウンロード可能。

🔧

自動アップデート

Sparkle によるアプリ自動更新。yt-dlp や FFmpeg もアプリ内でワンクリック更新。

対応サイト (一部)

YouTube ニコニコ動画 TVer ABEMA TikTok X (Twitter) Instagram Twitch Bilibili SoundCloud Bandcamp NHK Radiko Vimeo Dailymotion + 1800 以上

インストール方法

  1. DMG をダウンロード
    GitHub Releases から最新版を取得
  2. Applications にドラッグ
    DMG を開き、アプリを Applications フォルダにドラッグ
  3. 右クリック →「開く」で初回起動
    署名なし配布のため、初回は右クリックから開いてください。
    または: xattr -cr /Applications/yt-dlp-swift.app
